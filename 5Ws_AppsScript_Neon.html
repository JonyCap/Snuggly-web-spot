🔷 5Ws of Google Apps Script

1. Who uses it?

👤 Anyone working with Google tools like:

  • Students (e.g., you!)
  • Teachers
  • Office workers
  • Developers
  • Project managers

🔹 Example: A teacher uses Apps Script to send automatic feedback to students from a Google Sheet.

2. What is it?

🛠 A coding platform by Google (uses JavaScript) to:

  • Automate tasks
  • Connect and control Google services (Sheets, Docs, Drive, Gmail, Calendar, etc.)
  • Create add-ons, custom forms, web apps, or internal tools

🔹 Example: A student writes a script that auto-calculates course grades in a Google Sheet.

3. When is it used?

⏱ You use Apps Script:

  • When tasks are repetitive (e.g., sending 50 emails)
  • When Google Sheets or Docs need extra features
  • When building light web tools without using external servers

🔹 Example: Every morning, your script checks deadlines in a Google Sheet and sends you a reminder email.

4. Where is it used?

🌐 Directly inside Google Workspace:

  • Open Google Sheets > Extensions > Apps Script
  • Also runs as a Web App or can trigger actions like a scheduled job (cron-like)

🔹 Example: In your browser, you edit the script online — no downloads needed.

5. Why use it?

✅ It’s free, powerful, cloud-based, and connects with Google products easily.

Benefits:

  • Saves time
  • Increases productivity
  • Makes Google tools smarter
  • Easy to learn for beginners

🔹 Example: A company uses Apps Script to automatically send invoices to clients each month.

🔎 Real-World Examples (That You Could Do Too)

📌 Task ✅ What Apps Script Can Do
Grade Tracker Calculate averages and display your GPA
Email Sender Send custom email reports to classmates or team
To-Do Reminder Check Google Sheet tasks and send daily reminders
Attendance Tracker Log class attendance in Google Sheets and auto-notify absentees
Website Integration Fetch data from Sheets and display it on your HTML website
